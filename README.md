# website-trackmania_xjk

Monorepo for xjk websites and Trackmania backend services.

## Start Here

- `SETUP.md` - local/server setup and deployment guide
- `docs/README.md` - documentation index
- `services/README.md` - backend service map
- `services/ARCHITECTURE.md` - service boundaries and ownership

## Service Layout

- `services/altered` - community portal and project admin workflows
- `services/tracker` - map tracking runtime (`wr` and `leaderboard` modes)
- `services/tracker-displayname` - account ID to display-name tracking
- `services/tracker-club` - club/campaign/upload ingest runtime
- `services/aggregator` - shared cache API
